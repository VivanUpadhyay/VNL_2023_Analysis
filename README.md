#    VNL Men's 2023 Data Analysis

##  Project Overview

This project analyzes the Volleyball Nations League (VNL) Men's 2023 dataset using Python.

The objective is to discover player and team performance through exploratory data analysis and visualization.

---

##  Dataset

- Tournament: Volleyball Nations League (VNL) Men's 2023
- File: VNL2023.csv

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

##  Analysis Performed

- Data Cleaning
- Missing Value Detection
- Duplicate Removal
- Top Attackers
- Top Blockers
- Top Servers
- Team Performance
- Correlation Analysis
- Data Visualization

---

##  Key Insights

- Identified the best attacking players.
- Compared team performances.
- Analyzed relationships between different skills.
- Visualized trends using charts and heatmaps.

---

##  How to Run

1. Clone this repository

```bash
git clone https://github.com/VivanUapadhay/VNL_Men_2023_Analysis.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Open the notebook

```bash
jupyter notebook
```
